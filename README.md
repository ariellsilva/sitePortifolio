# 🌐 Ariell Site - Portfólio Pessoal

Este é um site pessoal desenvolvido por **Ariell Ferreira da Silva**, estudante de Ciência da Computação e desenvolvedora back-end em formação. O objetivo da página é apresentar um pouco sobre sua trajetória, experiências, cursos realizados e links para suas redes profissionais.

## 🖼️ Visão Geral

O site é composto por uma estrutura simples em **HTML e CSS**, com foco na clareza das informações e fácil navegação. Nele você encontrará:

- **Sobre mim**: Um resumo das experiências profissionais e acadêmicas da Ariell.
- **Projetos**: Link direto para o GitHub com os projetos desenvolvidos.
- **Contato**: Link para o LinkedIn da desenvolvedora.

## 🧠 Tecnologias Utilizadas

- HTML5
- CSS3
- Links externos (GitHub e LinkedIn)
